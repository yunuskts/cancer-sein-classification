## Diagnostic de cancer du sein 🎗️

Classification binaire (tumeur maligne/bénigne) — Kaggle Score : **0.989**

## Description
Pipeline ML complet sur données médicales (569 observations, 30 features).
Comparaison KNN, Decision Tree, SVM, Random Forest avec optimisation des hyperparamètres.
Priorité au recall pour minimiser les faux négatifs en contexte médical.

## Stack
Python · Scikit-learn · KNN · SVM · GridSearchCV · Cross-validation · StandardScaler

## Résultats
| Modèle | Score Kaggle |
| KNN baseline | 0.77 |
| KNN + StandardScaler | 0.97 |
| KNN optimisé (manhattan, k=3) | **0.989** |
